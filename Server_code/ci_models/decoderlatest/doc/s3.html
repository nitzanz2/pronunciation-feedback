Introduction

Sphinx-3 (S3) is the successor to the Sphinx-II speech recognition system from Carnegie Mellon University. The main differences between the two are:
  • S3 supports a much more flexible range of acoustic modelling. Specifically, it can handle discrete, semi-continuous, or fully continuous acoustic models. Sphinx-II can only handle semi-continuous models.
  • S3 is completely written from scratch and does not contain any historical appendages.

The Sphinx-3 Decoder Suite

The entire set of Sphinx-3 decoder programs can be found under: /afs/cs/project/plus-2/s3. Binaries for alpha and linux(x86) platforms are under bin/alpha and bin/linux respectively, and include the following:
  • s3decode: The Viterbi decoder using beam search.
  • s3dag: Shortest path search of the DAG constructed from the Viterbi decoder word lattice.
  • s3astar: N-best list generation from the Viterbi decoder word lattice.
  • s3align: Viterbi forced alignment.
  • s3allphone: Allphone Viterbi decoder.
Running any of these programs without any arguments produces a short description of the command line arguments (and their defaults) needed to run the program.

Example csh scripts can be found in: /afs/cs/project/plus-2/s3/bin.
